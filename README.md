# Bulletin Board Project

**Bulletin Board** — це сучасна веб-платформа для розміщення та керування оголошеннями. Проект розроблений на базі стеку .NET 9 та реалізує повноцінну архітектуру для роботи з даними, API та користувацьким інтерфейсом.

## Основні можливості
- **Керування оголошеннями**: повний цикл CRUD (створення, читання, оновлення, видалення).
- **Багаторівнева система категорій**: гнучка структура категорій та підкатегорій для зручного пошуку.
- **Динамічний інтерфейс**: використання AJAX для плавної взаємодії з користувачем без перезавантаження сторінок.
- **RESTful API**: відкритий інтерфейс для інтеграції з іншими сервісами.

## Технологічний стек
- **Backend**: .NET 9, ASP.NET Core Web API.
- **Frontend**: ASP.NET Core MVC, Bootstrap 5, Cyberpunk CSS.
- **Data Access**: Dapper (мікро-ORM) для високопродуктивної роботи з SQL Server.
- **Database**: Microsoft SQL Server із використанням збережених процедур.

## Архітектура проекту
Рішення розділене на логічні шари для забезпечення масштабованості та підтримки:
- **BulletinBoard.Web**: Веб-інтерфейс для кінцевих користувачів.
- **BulletinBoard.API**: Центральний вузол обробки даних та бізнес-логіки.
- **BulletinBoard.Core**: Спільні моделі, інтерфейси та доменна логіка.
- **BulletinBoard.Data**: Реалізація репозиторіїв та пряма взаємодія з базою даних.
- **BulletinBoard.Database**: Скрипти міграцій, структури таблиць та процедур.
